## 🔧 如果图片无法显示

### 检查步骤:

1. **检查网络连接**
   ```bash
   ping images.unsplash.com
   ```

2. **检查浏览器控制台**
   - 按F12打开开发者工具
   - 查看Network标签中的图片请求
   - 检查是否有404或其他错误

3. **测试单个图片URL**
   在浏览器中直接打开图片URL，例如：
   ```
   https://images.unsplash.com/photo-1490481651871-ab68de25d43d?w=800&h=600&fit=crop
   ```

4. **检查图片格式**
   - 确认URL参数格式正确
   - 确认没有URL编码问题

### 备选方案:

如果Unsplash无法访问，可以：

1. **使用本地图片**
   - 下载Unsplash图片到 `images/` 文件夹
   - 更新 `js/data.js` 中的图片路径

2. **使用其他CDN**
   - Pexels API
   - Pixabay
   - 自建图床

3. **使用占位符服务**
   - placeholder.com
   - via.placeholder.com
