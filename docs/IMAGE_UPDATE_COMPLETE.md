## 💡 Lorem Picsum 特点

### 随机数参数
`?random=X` 参数确保：
- 每个位置显示不同的图片
- 图片不会重复
- 可以通过数字控制显示的图片

### 尺寸灵活
可以随时调整图片尺寸：
```javascript
// 修改宽高即可
https://picsum.photos/1200/800?random=1  // 更大
https://picsum.photos/400/300?random=1   // 更小
```

### 图片缓存
Lorem Picsum自动缓存图片：
- 首次加载后会被浏览器缓存
- 后续访问速度更快
- 减少服务器负担

## 🔧 如果还有问题

### 问题1: 图片显示相同的内容
**原因**: 浏览器缓存  
**解决**: 
```
清除浏览器缓存
或使用不同的random数字
```

### 问题2: 图片加载慢
**原因**: 首次访问，图片需要生成  
**解决**: 
```
等待几秒，图片会被缓存
后续访问会很快
```

### 问题3: 某张图片不显示
**原因**: 极少数情况下random数字可能对应无效图片  
**解决**: 
```javascript
// 更换random数字
// 将 ?random=10 改为 ?random=50
```
