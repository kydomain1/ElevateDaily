## 🌐 Pexels 图片URL格式

所有图片使用统一的Pexels CDN格式：

```
https://images.pexels.com/photos/{ID}/pexels-photo-{ID}.jpeg?auto=compress&cs=tinysrgb&w={宽度}&h={高度}
```

### 参数说明：
- **auto=compress** - 自动压缩优化
- **cs=tinysrgb** - 色彩空间优化
- **w** - 宽度（800或400）
- **h** - 高度（600, 500或300）

## 💡 维护说明

### 如果需要更换某张图片

1. **访问Pexels**
   ```
   https://www.pexels.com
   ```

2. **搜索相关主题**
   - 例如：搜索 "minimalist fashion"
   - 选择合适的图片

3. **获取图片ID**
   - 图片URL中的数字就是ID
   - 例如：`/photos/1336873/` 中的 `1336873`

4. **替换URL**
   ```javascript
   // 在 js/data.js 中找到对应位置
   // 替换ID即可
   https://images.pexels.com/photos/新ID/pexels-photo-新ID.jpeg?auto=compress&cs=tinysrgb&w=800&h=600
   ```
